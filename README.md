# Каталоги произведений искусства

Данный сборник содержит каталоги работ некоторых уральских художниц. Я начал составлять их в мае 2025 г., с целью сохранить значимую информацию для истории. Я выложил их в качестве репозитория, чтобы было проще работать с нескольких компьютеров, а также, чтобы они не пропали. Идея данного проекта в документировании местного культурного наследия в виде распределенной системы каталогов, данные которых хранятся в виде XML-файлов в git-репозитории, а визуальное представление отображается в виде статического сайта на Github Pages. Если вы хотите присоединиться к этому проекту и вести каталог работ авторов из своего города, или, возможно, своих работ, см. данное [руководство](https://gitflic.ru/project/smallsoft/catalogtools/blob?file=docs%2Fguide.md&branch=main&mode=markdown).

В каталогах работ, если авторское название работы неизвестно, название придумано составителем и дано в круглых скобках. Аннотации к картинам приведены авторские (точная цитата или отредактированная с исправлением орфографии и пунктуации), если не указано иное. Когда год написания работы неизвестен, вместо него приведен год публикации репродукции. Комментарии от составителя следует воспринимать как любительские, написанные человеком, не имеющим образования в области искусства.
